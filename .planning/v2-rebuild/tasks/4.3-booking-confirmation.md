# Task 4.3 — Booking Confirmation + Email (Resend)

## Goal
Create the booking success page and send confirmation emails via Resend.

## Prerequisites
- Task 4.2 complete (booking flow exists)

## Steps

### 1. Install Resend
```bash
npm install resend
```

### 2. Resend Config
Create `lib/resend/config.ts`:
```ts
import { Resend } from 'resend'

export const resend = new Resend(process.env.RESEND_API_KEY!)
```

### 3. Email Templates
Create `lib/resend/emails.ts`:
- `sendBookingConfirmationToCustomer(booking, provider, service)` — send email to customer confirming their booking
  - Subject: "Your Rinse booking is confirmed"
  - Body: service name, provider name, date/time, location, amount paid
- `sendNewBookingToProvider(booking, customer, service)` — notify provider of new booking
  - Subject: "New booking from [Customer Name]"
  - Body: service requested, date/time, customer location, amount (minus platform fee)

Use simple HTML emails (no need for React Email templates at MVP stage):
```ts
await resend.emails.send({
  from: 'Rinse <bookings@rinse-it.com>', // or use Resend's test domain
  to: email,
  subject: '...',
  html: '...',
})
```

### 4. Success Page
Create `app/booking/[providerId]/success/page.tsx`:
- Server component
- Fetch booking details from URL param `?booking=[id]`
- Display confirmation card:
  - Checkmark icon (green)
  - "Booking Confirmed!"
  - Service name, provider name
  - Date and time
  - Location
  - Amount paid
  - Booking reference ID
- CTA buttons: "View My Bookings" → `/dashboard/customer`, "Back to Home" → `/`
- Trigger email sending (call a server action or API)

### 5. Send Emails on Booking Creation
Add email sending to the booking creation flow:
- Either in the `createBooking` action (after payment confirmation)
- Or create an API endpoint `app/api/bookings/confirm/route.ts` that:
  1. Verifies payment was successful
  2. Updates booking status to 'confirmed'
  3. Sends both emails

### 6. Design
- Success page: centered card with celebration feel
- Green checkmark, gold accents
- Clean booking summary

### 7. Verify
```bash
npm run build
```

## Output Files
- `lib/resend/config.ts`
- `lib/resend/emails.ts`
- `app/booking/[providerId]/success/page.tsx`
- `app/api/bookings/confirm/route.ts` (optional, or integrate into existing action)
