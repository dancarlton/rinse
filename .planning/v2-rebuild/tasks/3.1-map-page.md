# Task 3.1 — Map Page with Mapbox GL JS

## Goal
Create the search/discovery page with an interactive Mapbox map showing nearby providers.

## Prerequisites
- Wave 2 complete

## Context
Customers use this page to find detailers near them. The page has a map on one side and a list of providers on the other. Providers are shown as markers on the map.

## Steps

### 1. Install Mapbox Dependencies
```bash
npm install mapbox-gl react-map-gl
npm install -D @types/mapbox-gl
```

### 2. Map Component
Create `components/map/search-map.tsx` (client component):
- Use `react-map-gl` with `Map` component
- Dark map style: `mapbox://styles/mapbox/dark-v11`
- Access token from `process.env.NEXT_PUBLIC_MAPBOX_TOKEN`
- Default center: user's location (use browser Geolocation API) or fallback to a US center
- Default zoom: ~11 (city level)
- Show provider markers on the map using `Marker` components
- Each marker: gold colored pin/dot
- Click marker to show popup with provider name, rating, top service
- Popup has "View Profile" link to `/provider/[id]`

### 3. Search Page
Create `app/search/page.tsx`:
- Split layout: map on right (or top on mobile), provider list on left (or bottom on mobile)
- Search controls at top:
  - Location input (text, with "Use My Location" button)
  - Radius selector (5km, 10km, 25km, 50km dropdown)
  - Search button
- The page fetches nearby providers from the API (Task 3.2 will create the endpoint)
  - For now, create the fetch call to `/api/providers/nearby?lat=X&lng=X&radius=X`
  - Handle loading and empty states
- Pass providers to both the map and the list components

### 4. Map CSS
Add Mapbox GL CSS. In the search page or a layout, import:
```tsx
import 'mapbox-gl/dist/mapbox-gl.css'
```

### 5. Design
- Map fills the right side of the screen (desktop) or top half (mobile)
- Dark map style matches the app theme
- Gold markers for providers
- Provider list cards on the left/bottom
- Search bar with dark inputs, gold search button
- Responsive: stacked on mobile, side-by-side on desktop

### 6. Verify
```bash
npm run build
```
Build should pass even without Mapbox token (it's a client-side runtime dependency).

## Output Files
- `components/map/search-map.tsx`
- `app/search/page.tsx`
