# Task 1.1 — Next.js Scaffold + Project Structure

## Goal
Manually scaffold a Next.js 15 project on the `rinse-v2` orphan branch. Do NOT use `create-next-app` (it has interactive prompts).

## Current State
- Branch: `rinse-v2` (orphan, only `.claude/` and `.planning/` dirs exist)
- Working directory: `/Users/karlsmacbook/Documents/rinse`

## Steps

### 1. Initialize package.json
```bash
npm init -y
```
Update `package.json` with:
```json
{
  "name": "rinse",
  "version": "2.0.0",
  "private": true,
  "scripts": {
    "dev": "next dev --port 5050",
    "build": "next build",
    "start": "next start --port 5050",
    "lint": "next lint"
  }
}
```

### 2. Install Dependencies
```bash
npm install next@latest react@latest react-dom@latest
npm install -D typescript @types/node @types/react @types/react-dom eslint eslint-config-next
```

### 3. Create `tsconfig.json`
Standard Next.js TypeScript config with:
- `"target": "ES2017"`
- `"lib": ["dom", "dom.iterable", "esnext"]`
- `"moduleResolution": "bundler"`
- `"paths": { "@/*": ["./*"] }`

### 4. Create `next.config.ts`
```ts
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  // Config will grow as we add features
}

export default nextConfig
```

### 5. Create Directory Structure
```
app/
  layout.tsx        (root layout — minimal, just html/body/children)
  page.tsx          (placeholder: "Rinse v2 — Coming Soon")
  globals.css       (empty for now, Wave 1.2 will populate)
lib/                (empty — will hold supabase, stripe, etc.)
components/         (empty — will hold shared components)
  ui/               (empty — shadcn components go here)
types/              (empty — will hold database.ts etc.)
public/             (empty — static assets)
```

### 6. Root Layout (`app/layout.tsx`)
```tsx
import type { Metadata } from 'next'
import './globals.css'

export const metadata: Metadata = {
  title: 'Rinse — Mobile Car Care',
  description: 'On-demand mobile car wash and detailing',
}

export default function RootLayout({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  )
}
```

### 7. Placeholder Page (`app/page.tsx`)
Simple centered text: "Rinse v2 — Coming Soon"

### 8. Create `.gitignore`
Standard Next.js gitignore:
```
node_modules/
.next/
out/
.env*.local
```

### 9. Verify
```bash
npm run build
```
Must pass with zero errors.
