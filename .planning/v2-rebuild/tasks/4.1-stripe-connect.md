# Task 4.1 — Stripe Connect Onboarding for Providers

## Goal
Set up Stripe Connect so providers can onboard their accounts and receive payments (minus 15% platform fee).

## Prerequisites
- Wave 2 complete (provider dashboard exists)

## Steps

### 1. Install Stripe
```bash
npm install stripe
```

### 2. Stripe Config
Create `lib/stripe/config.ts`:
```ts
import Stripe from 'stripe'

export const stripe = new Stripe(process.env.STRIPE_SECRET_KEY!, {
  apiVersion: '2024-12-18.acacia', // use latest stable
})

export const PLATFORM_FEE_PERCENT = 15
```

### 3. Stripe Connect Account Creation API
Create `app/api/stripe/connect/route.ts`:
- POST: Create a Stripe Connect Express account for the provider
  - Set `type: 'express'`
  - Store `stripe_account_id` in provider's profile
  - Return an account link URL for onboarding

```ts
// Create Express account
const account = await stripe.accounts.create({
  type: 'express',
  email: user.email,
  metadata: { supabase_user_id: user.id },
})

// Save to profile
await supabase.from('profiles').update({ stripe_account_id: account.id }).eq('id', user.id)

// Create onboarding link
const accountLink = await stripe.accountLinks.create({
  account: account.id,
  refresh_url: `${origin}/dashboard/provider/stripe?refresh=true`,
  return_url: `${origin}/dashboard/provider/stripe?success=true`,
  type: 'account_onboarding',
})

return NextResponse.json({ url: accountLink.url })
```

### 4. Stripe Connect Webhook (Optional for now)
Create `app/api/stripe/webhook/route.ts`:
- Handle `account.updated` events to mark `stripe_onboarding_complete = true`
- Parse webhook signature for security
- For MVP: we can also check onboarding status on the Stripe page load

### 5. Provider Stripe Dashboard Page
Create `app/dashboard/provider/stripe/page.tsx`:
- Server component — check if provider has Stripe account
- States:
  1. No Stripe account: "Connect with Stripe to receive payments" + "Set Up Payments" button
  2. Account exists but onboarding incomplete: "Complete your Stripe setup" + "Continue Setup" button
  3. Onboarding complete: "Payments active" with green status, link to Stripe Express dashboard
- Check onboarding status by querying Stripe:
  ```ts
  const account = await stripe.accounts.retrieve(profile.stripe_account_id)
  const isComplete = account.charges_enabled && account.payouts_enabled
  ```

### 6. Verify
```bash
npm run build
```
