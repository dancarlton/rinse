# Task 1.5 — Layout + Navbar + Landing Page

## Goal
Build the root layout with a responsive navbar, footer, and a compelling landing page that communicates Rinse's value proposition.

## Prerequisites
- Task 1.1 complete (Next.js scaffold)
- Task 1.2 complete (Tailwind + shadcn/ui theme)
- Task 1.3 complete (Supabase client setup)

## Steps

### 1. Update Root Layout
`app/layout.tsx`:
- Import the Inter font from `next/font/google`
- Apply dark class to `<html>`
- Wrap children with Navbar and Footer
- Set metadata: title "Rinse — Mobile Car Care", description, etc.

```tsx
import { Inter } from 'next/font/google'
const inter = Inter({ subsets: ['latin'] })
```

### 2. Navbar Component
`components/navbar.tsx`:
- Client component (needs auth state + mobile menu toggle)
- Fixed/sticky at top, dark background with slight transparency/blur
- Logo: "RINSE" text in gold, bold
- Navigation links: Home, Find a Detailer, How It Works
- Auth state:
  - Not logged in: "Sign In" link, "Get Started" gold button
  - Logged in: user avatar/name dropdown with "Dashboard" and "Sign Out"
- Use `createClient()` from `@/lib/supabase/client` to check auth state
- Mobile: hamburger menu that opens a slide-down/slide-in menu
- Use shadcn Button component

### 3. Footer Component
`components/footer.tsx`:
- Simple footer with dark background
- "© 2026 Rinse. All rights reserved."
- Links: About, Privacy, Terms (can be `#` placeholders for now)
- Subtle gold accent line at top of footer

### 4. Landing Page
`app/page.tsx`:
Full landing page with these sections:

**Hero Section:**
- Large headline: "Premium Car Care, Delivered to You" or similar
- Subheadline explaining mobile detailing on-demand
- CTA button: "Find a Detailer Near You" → links to `/search` (gold button, large)
- Secondary CTA: "Become a Provider" → links to `/register`
- Dark background with subtle gradient or pattern

**How It Works Section:**
- 3 steps with icons:
  1. "Find" — Search for detailers near your location
  2. "Book" — Choose a service and schedule a time
  3. "Relax" — Your detailer comes to you
- Each step in a card with icon, title, description
- Gold accent on step numbers or icons

**For Providers Section:**
- "Grow Your Detailing Business"
- Benefits: Set your own prices, manage your schedule, get paid directly
- CTA: "Start Earning" → `/register`

**Simple Footer/Bottom CTA:**
- "Ready to get started?" with sign-up button

### 5. Design Notes
- Use gold (#D4AF37) sparingly — primary CTAs, accents, logo
- Near-black background throughout (#0A0A0A)
- Cards use surface color (#141414)
- Large readable typography
- Sections separated with subtle borders or spacing
- Icons can use Lucide React (already installed with shadcn)
- Keep it clean and premium-feeling — not cluttered

### 6. Verify
```bash
npm run build
```
Must pass.

## Output Files
- `app/layout.tsx` (updated)
- `app/page.tsx` (full landing page)
- `components/navbar.tsx`
