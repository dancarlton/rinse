# Task 3.2 — Geospatial Search API (PostGIS)

## Goal
Create the API endpoint that searches for nearby providers using PostGIS geospatial queries.

## Prerequisites
- Wave 1 complete (Supabase setup with PostGIS schema)

## Steps

### 1. Nearby Providers API
Create `app/api/providers/nearby/route.ts`:

```ts
import { createClient } from '@/lib/supabase/server'
import { NextRequest, NextResponse } from 'next/server'

export async function GET(request: NextRequest) {
  const searchParams = request.nextUrl.searchParams
  const lat = parseFloat(searchParams.get('lat') || '0')
  const lng = parseFloat(searchParams.get('lng') || '0')
  const radius = parseFloat(searchParams.get('radius') || '25') // km

  if (!lat || !lng) {
    return NextResponse.json({ error: 'lat and lng are required' }, { status: 400 })
  }

  const supabase = await createClient()

  // Call the PostGIS function defined in migration 001
  const { data, error } = await supabase.rpc('find_nearby_providers', {
    lat,
    lng,
    radius_km: radius,
  })

  if (error) {
    return NextResponse.json({ error: error.message }, { status: 500 })
  }

  return NextResponse.json({ providers: data })
}
```
