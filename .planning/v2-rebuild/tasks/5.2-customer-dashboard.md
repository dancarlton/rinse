# Task 5.2 — Customer Dashboard (Bookings, Leave Reviews)

## Goal
Create the customer dashboard showing their bookings and ability to leave reviews.

## Prerequisites
- Wave 4 complete (bookings exist)

## Steps

### 1. Customer Dashboard Layout
Create `app/dashboard/customer/layout.tsx`:
- Simple layout with sidebar or top nav
- Links: My Bookings, Find a Detailer, Settings
- Dark theme

### 2. Customer Dashboard Page
Create `app/dashboard/customer/page.tsx`:
- Server component — fetch customer's bookings
- Auth guard

### 3. Dashboard Sections

**Upcoming Bookings:**
- Confirmed/pending bookings sorted by date
- Show: provider name, service, date/time, status badge
- "Cancel" button for pending/confirmed bookings
- "View Provider" link

**Past Bookings:**
- Completed bookings
- Show: provider name, service, date, amount paid
- "Leave Review" button (if no review exists for that booking)
- "View Review" if review already left

**Quick Stats:**
- Total bookings
- Money spent
- Providers used

### 4. Review Form
Create `components/customer/review-form.tsx` (client component):
- Dialog/modal that opens when "Leave Review" is clicked
- Star rating selector (1-5, clickable stars)
- Comment textarea
- Submit button
- Uses server action to create review

### 5. Review Server Actions
Add to `app/actions/bookings.ts` or create `app/actions/reviews.ts`:
- `createReview(bookingId: string, rating: number, comment: string)` — create review
  - Verify booking belongs to customer and is completed
  - Verify no existing review for this booking
- `getReviewsForProvider(providerId: string)` — fetch reviews

### 6. Dashboard Route Logic
Update `app/dashboard/page.tsx`:
- Instead of a generic placeholder, redirect based on user role:
  - Provider → `/dashboard/provider`
  - Customer → `/dashboard/customer`

### 7. Design
- Booking cards with status badges
- Star rating selector with gold filled stars
- Clean review modal
- Responsive layout

### 8. Verify
```bash
npm run build
```

## Output Files
- `app/dashboard/customer/layout.tsx`
- `app/dashboard/customer/page.tsx`
- `app/dashboard/page.tsx` (updated with role redirect)
- `components/customer/review-form.tsx`
- `app/actions/reviews.ts`
