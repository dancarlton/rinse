# Task 1.2 — Tailwind CSS + shadcn/ui Dark Premium Theme

## Goal
Install Tailwind CSS v4 and shadcn/ui. Configure a dark premium theme with near-black backgrounds and gold accents.

## Prerequisites
- Task 1.1 complete (Next.js scaffolded, `package.json` exists)

## Steps

### 1. Install Tailwind CSS v4 for Next.js
```bash
npm install tailwindcss @tailwindcss/postcss postcss
```

Create `postcss.config.mjs`:
```js
const config = {
  plugins: {
    "@tailwindcss/postcss": {},
  },
};
export default config;
```

Update `app/globals.css`:
```css
@import "tailwindcss";
```

### 2. Initialize shadcn/ui
```bash
npx shadcn@latest init
```
When prompted:
- Style: New York
- Base color: Neutral
- CSS variables: Yes

If interactive prompts block, create `components.json` manually:
```json
{
  "$schema": "https://ui.shadcn.com/schema.json",
  "style": "new-york",
  "rsc": true,
  "tsx": true,
  "tailwind": {
    "config": "",
    "css": "app/globals.css",
    "baseColor": "neutral",
    "cssVariables": true
  },
  "aliases": {
    "components": "@/components",
    "utils": "@/lib/utils",
    "ui": "@/components/ui",
    "lib": "@/lib",
    "hooks": "@/hooks"
  },
  "iconLibrary": "lucide"
}
```

### 3. Install shadcn utility
```bash
npm install clsx tailwind-merge class-variance-authority lucide-react
```

Create `lib/utils.ts`:
```ts
import { type ClassValue, clsx } from "clsx"
import { twMerge } from "tailwind-merge"

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}
```

### 4. Dark Premium Theme
Update `app/globals.css` with CSS variables for the dark theme:

**Color palette:**
- Background: `#0A0A0A` (near-black)
- Card/surface: `#141414`
- Muted: `#1F1F1F`
- Border: `#2A2A2A`
- Primary/accent: `#D4AF37` (gold)
- Primary foreground: `#0A0A0A` (dark text on gold)
- Text: `#FAFAFA` (almost white)
- Muted text: `#A1A1AA`
- Destructive: `#EF4444`

Set up both `:root` (dark as default) with proper HSL CSS variables that shadcn expects.

Add to body: `antialiased` font smoothing, dark background, light text.

### 5. Install a Few Base shadcn Components
```bash
npx shadcn@latest add button card input label
```
These are needed by almost every page.

### 6. Verify
```bash
npm run build
```
Must pass. The app should render with the dark theme when running `npm run dev`.

## Output Files
- `postcss.config.mjs`
- `app/globals.css` (updated with Tailwind + theme variables)
- `components.json`
- `lib/utils.ts`
- `components/ui/button.tsx`
- `components/ui/card.tsx`
- `components/ui/input.tsx`
- `components/ui/label.tsx`

## Design Reference
- Near-black backgrounds (#0A0A0A)
- Gold accent (#D4AF37) for primary buttons, links, highlights
- Clean modern typography (system font stack or Inter)
- All shadcn components should look correct in dark mode
